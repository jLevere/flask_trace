Trace service

This webapp primarily returns the headers of your request plain text and json.

The request ip is always provided in the response headers as 'remote_addr'. Some X headers may be attached by the webapp proxy, but will not overwrite existing headers, only appends to them with a comma.

The provided pages are: Json headers, plain text headers, and plain text request ip
